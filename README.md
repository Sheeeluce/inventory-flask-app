# Inventory Management System

## Description

A Flask REST API that allows users to manage inventory items. Product details are retrieved automatically from the OpenFoodFacts API using a barcode.

## Features

- View all inventory items
- View one inventory item
- Add inventory items
- Update inventory items
- Delete inventory items
- OpenFoodFacts API integration
- CLI interface
- Unit testing with pytest

## Technologies

- Python
- Flask
- Requests
- Pytest

## Installation

```bash
pip install flask requests pytest
```

## Run

```bash
python app.py
```

## Run CLI

```bash
python cli.py
```

## Run Tests

```bash
pytest
```
